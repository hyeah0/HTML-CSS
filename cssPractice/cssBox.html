width: 300px;
height: 300px;
padding: 20px 40px;
box-sizing: border-box;
ㄴ box-sizing을 사용함으로
가로 300px, 세로 300px 설정값인 네모박스

CSS선언시 맨처음은 하단과 같이 많이 선언 

                    *{
                        box-sizing: border-box;
                    }

Block Inline InlineBlock Flex

block

따로 width를 설정하지 않은 경우 : 부모의 content-box의 가로길이
width를 설정한경우 : 남은 공간은 margin으로 채움

테스트1

테스트2 : width 설정

테스트3 : 가운데위치
width 값 먼저 설정
margin: auto;
margin-left: auto; (빈공간을 왼쪽에)
