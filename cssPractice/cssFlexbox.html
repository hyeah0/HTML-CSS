Flexbox

            .flexbox{
                display: flex;
                flex-direction: row;
                flex-wrap : nowrap;
            }
  1. display : flex; 부모요소에 기재

  2. flex-direction: row; -> 가로방향
    flex-direction: column; -> 세로방향

    main axis => 메인이 되는 축
    flex-direction: row; 일때는 x축(왼쪽에서 오른쪽으로)
    flex-direction: column; 일때는 y축(위에서 아래쪽으로)
    justify-content: center (가운데 정렬) justify-content: flex-end (오른쪽 정렬) justify-content: center 가운데 정렬

    cross axis => 메인이 되는축에 90도 지나는 선
    align-items align-content
  3. flex-wrap : nowrap -> 한줄정렬(자식의 크기를 줄여서라도 한줄정렬)
    flex-wrap : wrap -> 묶어서 정렬
