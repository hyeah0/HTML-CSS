Flexbox

            .flexbox{
                display: flex;
                flex-direction: row;
                justify-content: center;
                align-items: center; 또는
                align-content: center; 
                flex-wrap : nowrap;
            }
  1. display : flex; 부모요소에 기재

  2. flex-direction: row; -> 가로방향
    flex-direction: column; -> 세로방향

    main axis => 메인이 되는 축
    flex-direction: row; 일때는 x축(왼쪽에서 오른쪽으로)
    flex-direction: column; 일때는 y축(위에서 아래쪽으로)

    cross axis => 메인이 되는축에 90도 지나는 선

  3. justify-content: center; (가운데 정렬)
    justify-content: flex-end; (오른쪽 정렬)
    justify-content: space-between; (요소의 간격을 동일하게 변경)
    justify-content: space-around;

    align-items: center; (하나의 cross axis 기준에서 가운데)
    align-items: flex-start;(하나의 cross axis 기준에서 윗쪽)
    align-items: flex-end;(하나의 cross axis 기준에서 끝)

    align-content: center; (전체의 cross axis 기준으로 센터)
    flex-wrap : wrap; 일때 사용

  4. flex-wrap : nowrap -> 한줄정렬(자식의 크기를 줄여서라도 한줄정렬)
    flex-wrap : wrap -> 칸이 모자르면 하단에 정렬

space-evenly
The "space-evenly" value for the justify-content property distributes the space between items evenly. It is similar to space-around but provides equal instead of half-sized space on the edges. Can be used in both CSS flexbox & grid.
