Transition

효과를 줄 때 : 속성의 값이 변화할때(예시 : 클릭을 할때)

  • property
  • duration : 지속시간(단위 : 1000ms= 1s)/ 변경시간
  • timing-funcion : 변화 속도 (생략가능)
    • ease-in : 천천히 변경되다가 나중에 빨리 변경
    • ease-out : 빨리 변경되다가 나중에 천천히 변경
    • ease-in-out
    • cubic-bezier
  • delay 
                            .box1 {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 215, 222);
                                position: relative;
                              
                                transition: background-color 3s, left 5s;
                                /* transition: all 2.5s; */
                              }
                              
                              .box1:hover {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 0, 0);
                              
                                left: 50px;
                            }
timing-funcion , ease-in
                            .box2 {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 215, 222);
                                transition: all 3s ease-in;
                            }
                            
                            .box2:hover {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 0, 0);
                            }
cubic-bezier
                            .box3 {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 215, 222);
                                transition: all 6s cubic-bezier(0.3, 1.09, 0.93, 0.53);
                            }
                            
                            .box3:hover {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 0, 0);
                            }
delay
                            .box4 {
                                width: 80px;
                                height: 80px;
                                background-color: rgb(255, 215, 222);
                                transition: all 1s 2s;
                            }
                            
                            .box4:hover {
                                width: 80px;

                                background-color: rgb(255, 0, 0);
                            }

animation

효과를 줄 때 : 속성 값 변화 없이 효과를 주고 싶을때

                        @keyframe name{
                                        from {
                                            border:1px solid black;
                                        }
                                        to{
                                            background-color: blue;
                                        }
                                    }

from : 시작할때 대신 0% 사용가능

to : 종료할때 대신 1%~100% 사용가능

애니메이션 박스 예시1 
                            .animationExemple1 {
                            width: 100px;
                            height: 100px;
                            background-color: black;
                            color: white;
                        
                            display: flex;
                            justify-content: center;
                            align-items: center;
                            text-align: center;
                        
                            
                            animation-name: rotate;
                            animation-duration: 3s;
                            animation-timing-function: ease-in-out;
                            animation-direction: alternate;
                            /* alternate : 번갈아가면서 실행 */
                            animation-iteration-count: 3; 
                            
                               
                            }
                            
                            @keyframes rotate {
                                0% {
                                background-color: black;
                                }
                                50% {
                                background-color: orange;
                                transform: rotate(10deg);
                                }
                                100% {
                                background-color: rgb(118, 247, 240);
                                border-radius: 50%;
                                }
                            }
