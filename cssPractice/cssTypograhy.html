Typograhy = 텍스트 디자인

  • font-size : 글자 크기
  • font-weight : 글자 굵기
    100 thin / thin ltalic / font-weight: 100;
    300 light/light ltalic / font-weight: 300;
    400 regular/regular ltalic / font-weight: 400;
    500 medium/medium ltalic / font-weight: 500;
    700 blod/bold ltalic / font-weight: 700;
    900 black/black ltalic / font-weight: 900;
  • font-family : 서체
    font-family : "Poppins";
    font-family : "Poppins", sans-serif 또는 serif;
    ㄴPoppins 서체가 없으면 sans-serif 또는 serif(삐침이 있는 서체)서체 사용
    font-family : "Poppins","Roboto",sans-serif;
    ㄴPoppins 서체가 없으면 Roboto 사용, 두 서체다 없으면 sans-serif 서체 사용
  • color : 글씨 색상
    사용가능 단위
    hex 
    color: #ff9900;
    rgb 
    color: rgb(108, 108, 199);
    rgba 
    color: rgba(124, 45, 134, 0.411);
  • line-height : 줄간격
    단위: em은 em 표시 생략가능
    line-height : 1.5;
  • letter-spacing : 자간
    단위: px, em 사용가능(em 표시 생략 불가능)
  • 단위
    px
    em : 대문자 크기의 기준 (실제 적용 폰트사이즈 1em)
    24px(현재 적용된 폰트사이즈) = 1em
    1px(현재 적용된 폰트사이즈) = 1em
    rem : html에 적용된 폰트사이즈를 1rem
