mediaQuery

반응형 웹

필요
html : viewport meta
css : media query

html

            <head>
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>

css

            @media screen and (min-width: 700px){
                스타일
            }
            ㄴ 700px 이상에서는 { } 안의 스타일

예제

css문

                .box::after {
                    content: "media";
                  }
                  
                @media screen and (min-width: 576px){
                    .box {
                      background-color: rgb(159, 211, 189);
                    }
                    .box::after {
                      content: "phone";
                    }
                  }
                  
                @media screen and (min-width: 768px){
                    .box {
                      background-color: rgb(159, 162, 211);
                    }
                    .box::after {
                      content: "Tablet";
                    }
                  }
                  
                @media screen and (min-width: 768px) and (max-width: 991px){
                    .box {
                      background-color: rgb(209, 161, 171);
                    }
                    .box::after {
                      content: "max-width";
                    }
                  }
                  
                @media screen and (min-width: 992px){
                    .box {
                      background-color: rgb(186, 159, 211);
                    }
                    .box::after {
                      content: "Desktop";
                    }
                  }
                  
                @media screen and (min-width: 1200px){
                    .box {
                      background-color: rgb(207, 159, 211);
                    }
                    .box::after {
                      content: "Large Desktop";
                    }
                  }
