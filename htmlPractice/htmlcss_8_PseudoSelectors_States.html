states

hover, Active, focus, focus-within, visited

:hover
마우스가 해당 요소 위에 있을때
                    .hover button:hover {
                        background-color: pink;
                        }
:active
해당 요소를 클릭했을때
                    .active button:active {
                        background-color: yellow;
                      }
:focus
요소가 선택되었을때
                    .focus button:focus {
                        background-color: green;
                      }
:focus-within
부모요소에 적용하며, 자식요소 중 하나가 선택되었을때
                    html
                    <form>
                        <input type="text"/>
                        <input type="text"/>
                        <input type="text"/>
                    </form>
        
                    css
                    .focus_within form:focus-within {
                        background-color: violet;
                      }
:visited
링크가 방문한 사이트일 경우
넷플릭스
                    .visited a:visited {
                        color: tomato;
                      }
:복합으로 사용
                    html
                    <form>
                        <input type="text"/>
                        <input type="text"/>
                        <input type="text"/>
                    </form>
        
                    css
                    .mixed form:hover input:nth-child(2) {
                        background-color: orangered;
                      }
                    form에 마우스 커서가 올라가있을때 2번째 input에 style 적용
        
                    .mixed form:hover input:nth-child(3):focus {
                        background-color: pink;
                      }
                    form에 마우스 커서가 올라가 있고, 3번째 input이 focus 되었을때 style 적용
